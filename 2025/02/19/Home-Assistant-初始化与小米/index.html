Home Assistant 初始化与小米
2025-02-22 01:42:41 # 智能家居 #HomeAssistant #小米 #Docker #NAS
  • Home Assistant 官网
  • Xiaomi Home Integration for Home Assistant

1 部署 HA

以 Docker 部署为例。

镜像名称：homeassistant/home-assistant

  1. 部署 HA

    修改：/PATH_TO_YOUR_CONFIG

    群晖则在存储空间设置，这里我自定的目录为 /docker/homeassistant，可以按自己喜好修改。

  2. 访问 {{部署机 IP}}:8123，左下角可更改语言进行配置

  3. 进入刚刚我们修改 /PATH_TO_YOUR_CONFIG

  4. 新建 custom_components

2 安装 ha_xiaomi_home

写在前面：建议通过路由器给每个智能家居分配静态的 IP 地址，避免 IP 地址的变动导致配置失效。

  1. 下载 ha_xiaomi_home，这里用 git

  2. 把 ha_xiaomi_home 里的 xiaomi_home 目录复制到我们刚刚自己新建的 custom_components 目录下

  3. 重启 HA

  4. 设置 ➡️ 设备与服务 ➡️ 添加集成 ➡️ Xiaomi Home

  5. 登录你的小米账号

    可能出现无法跳转回 HA 的情况，此时在浏览器地址栏手动替换 homeassistant.local:8123 ➡️ {{部署机 IP}}:8123 后，回车即可。

  6. 导入米家设备至 HA

  7. 尝试使用设备验证是否配置成功

3 配置 HomeKit

  1. 设置 ➡️ 设备与服务 ➡️ 添加集成 ➡️ Apple ➡️ HomeKit Bridge

  2. 左下角通知查看二维码

  3. 打开手机 家庭 APP 扫码配置
