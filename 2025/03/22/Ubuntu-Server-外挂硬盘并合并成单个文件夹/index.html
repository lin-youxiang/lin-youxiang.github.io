📌 额外说明：USB 硬盘盒自动睡眠的问题

USB 硬盘盒一般会在一定时间（比如10分钟）没有读写后自动进入睡眠状态：

  • 优点：降低功耗、减少机械磨损。
  • 缺点：首次访问硬盘有轻微延迟（几秒钟），频繁唤醒可能降低硬盘寿命。

大部分情况下不用担心。如需检查硬盘启动次数：

每天几十次内启停属于正常范围，正常使用不会有影响。

🚀 总结与感想

本次实践：

  • 使用 GPT + EXT4 格式化磁盘；
  • 使用 mergerfs 工具灵活合并磁盘；
  • Docker 容器无感知挂载大容量存储空间。

整体操作安全简单，对硬盘数据不破坏，日后也可以随时灵活拆分。
